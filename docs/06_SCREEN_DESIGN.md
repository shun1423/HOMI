# 화면 설계

## 글로벌 레이아웃

```
┌─────────────────────────────────────────────┐
│  🏠 집짓기 에이전트        [사용자] [알림]    │
├──────────┬──────────────────────────────────┤
│          │                                  │
│ 사이드바  │          메인 콘텐츠              │
│          │                                  │
│ 📊 대시보드│                                  │
│ 📋 보드   │                                  │
│ 💬 채팅   │                                  │
│ 📍 장소   │                                  │
│ 👷 시공업자│                                  │
│ ⚖️ 비교   │                                  │
│ 🎨 영감   │                                  │
│ 📅 공정   │                                  │
│ 💰 비용   │                                  │
│ 📸 사진   │                                  │
│ 📄 문서   │                                  │
│ 🔗 Notion │                                  │
│          │                                  │
├──────────┴──────────────────────────────────┤
│  💬 빠른 채팅 입력 (항상 하단에 노출)          │
└─────────────────────────────────────────────┘
```

- 사이드바: 접기/펼치기 가능 (모바일에서는 햄버거 메뉴)
- 하단 빠른 채팅: 어느 화면에서든 대화형 입력 가능 (옵션)
- 모바일: 하단 탭 네비게이션으로 전환

## 모바일 레이아웃

- 하단 탭: 대시보드 / 보드 / 채팅 / 장소 / 더보기
- "더보기"에서 나머지 기능 접근
- 채팅은 풀스크린
- 사진 촬영 → 바로 업로드 (시공 사진, 영수증)
- 카메라 접근 허용 필요
