# 기능 상세 명세서

## F01. 프로젝트 보드

모든 기능의 중심. 공간별 × 항목별 매트릭스로 인테리어 결정사항을 관리.

### 핵심 요구사항

- **공간 관리**: 화장실, 거실, 안방, 부엌, 현관 등 사용자가 공간을 자유롭게 추가/편집/삭제
- **항목 관리**: 각 공간 하위에 바닥, 벽, 천장, 조명, 설비 등 항목 추가/편집/삭제
- **각 항목(셀)에 포함되는 정보**:
  - 상태: 미결정 / 후보 있음 / 결정됨 / 구매완료 / 시공완료
  - 결정 내용 (텍스트)
  - 후보 목록 (제품 비교 보드 F08과 연결)
  - 메모
  - 첨부 사진/파일
  - 관련 쇼룸/매장 링크 (장소 탐색 F02와 연결)
  - 예산/실제 비용 (비용 관리 F06과 연결)
- **뷰 모드**:
  - 매트릭스 뷰: 공간(행) × 항목(열) 테이블
  - 공간별 상세 뷰: 하나의 공간에 대한 모든 항목 카드 형태
  - 전체 진행률 대시보드
- **입력 방식**:
  - 대화형: 채팅(F04)에서 입력하면 자동 반영
  - 직접 입력: 셀 클릭 → 편집

---

## F02. 장소 탐색 + 스케줄

결정사항/후보 기반으로 쇼룸, 매장, 전시장을 찾고 방문 계획을 세움.

### 핵심 요구사항

- **장소 검색**:
  - 키워드 검색: "분청 세면대 쇼룸", "타일 매장 경기도"
  - 항목 기반 자동 추천: 프로젝트 보드에서 "화장실 세면대" 항목 선택 → 관련 쇼룸 자동 검색
  - LLM이 검색 키워드 자동 생성 (결정사항 기반)
- **장소 정보**:
  - 이름, 주소, 전화번호, 영업시간
  - 현재 위치(포천) 기준 거리/소요시간
  - 지도 표시
  - 웹사이트/블로그 링크
  - 메모 추가 가능
- **스케줄 짜기**:
  - 여러 장소 선택 → 최적 동선 자동 계산
  - 하루 일정 생성: 시간대별 방문 계획
  - 전화 예약 필요한 곳 표시
  - 캘린더 내보내기 (Google Calendar 등)
- **방문 후 기록**:
  - 방문 완료 체크
  - 사진 첨부
  - 메모/평가 남기기
  - 프로젝트 보드 항목과 연결

## F03. 프로젝트 관리

전체 인테리어/시공 공정을 타임라인으로 관리.

### 핵심 요구사항

- **공정 단계 관리**:
  - 기획 → 자재 선정 → 구매 → 철거 → 시공 → 마감 → 입주 등
  - 단계별 하위 태스크 (예: "시공" → "배관", "전기", "타일", "도배" 등)
  - 각 태스크: 담당자, 시작일, 종료일, 상태, 메모
- **뷰 모드**:
  - 칸반 보드: 상태별 (대기/진행중/완료)
  - 타임라인/간트차트: 일정 겹침 확인
  - 리스트 뷰
- **의존성**: "배관 완료 후 타일 시공" 같은 선후관계 설정
- **알림**: 다가오는 일정 리마인더

## F04. Q&A + 대화형 입력

하나의 채팅 인터페이스에서 질문도 하고, 결정사항 입력도 하는 AI 채팅.

### 핵심 요구사항

- **대화형 입력 (Intent: UPDATE)**:
  - "화장실 바닥은 마이크로 시멘트로 결정했어" → 프로젝트 보드 F01 자동 업데이트
  - "세면대 후보에 분청 도자기 추가해줘" → 비교 보드 F08에 추가
  - "어제 타일 매장 갔는데 A타일 30만원이었어" → 비용 관리 F06에 반영
  - LLM이 intent 파악 → 해당 기능에 반영 → 사용자에게 확인
- **Q&A (Intent: QUESTION)**:
  - "건식 화장실이랑 습식 차이가 뭐야?"
  - "마이크로 시멘트 유지보수 어떻게 해?"
  - "한옥에 어울리는 조명 스타일 추천해줘"
  - 웹 검색 + LLM 지식 기반 답변
- **Notion 분석 (Intent: NOTION_ANALYZE)**:
  - "노션에 있는 화장실 관련 내용 정리해줘"
  - Notion에서 데이터 가져와서 LLM이 분석/요약
- **채팅 이력**: 대화 내역 저장, 검색 가능
- **컨텍스트 유지**: 프로젝트 보드 현재 상태를 컨텍스트로 활용

## F05. Notion 연동

Notion과 양방향 데이터 싱크 + Notion 콘텐츠 AI 분석.

### 핵심 요구사항

- **내보내기 (앱 → Notion)**:
  - 프로젝트 보드 내용을 Notion DB로 내보내기
  - 비용 정리, 시공업자 정보 등도 내보내기 가능
  - 수동 트리거 (버튼) 또는 자동 싱크
- **가져오기 (Notion → 앱)**:
  - Notion 페이지/DB 선택해서 데이터 가져오기
  - 가져온 데이터를 프로젝트 보드에 매핑
- **AI 분석**:
  - Notion에 있는 메모, 회의록 등을 LLM이 읽어서 분석
  - "노션에 있는 인테리어 관련 내용 요약해줘"
  - "노션에서 아직 결정 안 된 항목들 뽑아줘"
  - 분석 결과를 프로젝트 보드에 반영 가능
- **연결 설정**: Notion OAuth 인증, 워크스페이스/페이지 선택 UI

## F06. 비용 관리

항목별 예산 및 실제 지출 추적. 영수증 OCR 자동 인식.

### 핵심 요구사항

- **예산 설정**: 공간별/항목별 예산 금액 설정
- **지출 기록**:
  - 수동 입력: 금액, 날짜, 업체, 카테고리, 메모
  - 대화형 입력: "타일 30만원 결제했어" → 자동 기록
  - **영수증 OCR**: 사진 촬영/업로드 → Claude Vision으로 금액, 날짜, 업체명 자동 추출 → 확인 후 저장
- **대시보드**:
  - 공간별 예산 vs 실지출 비교 차트
  - 전체 총 예산 vs 총 지출
  - 카테고리별 지출 비율
  - 최근 지출 내역 리스트
- **프로젝트 보드 연결**: 각 항목별 비용이 프로젝트 보드 셀에도 표시

---

## F07. 시공 사진 기록

공간별 전/중/후 사진을 타임라인으로 기록.

### 핵심 요구사항

- **사진 업로드**: 카메라 촬영 또는 갤러리에서 업로드
- **메타데이터**: 공간, 항목, 날짜, 시공 단계(전/중/후), 메모
- **뷰 모드**:
  - 타임라인: 날짜순 사진 스크롤
  - 공간별 필터: "화장실만 보기"
  - 비교 뷰: 전/후 나란히 보기
- **Supabase Storage**: 사진 파일 저장
- **프로젝트 보드 연결**: 해당 항목에서 관련 사진 바로 조회

---

## F08. 자재/제품 비교 보드

후보 제품들을 나란히 놓고 비교하여 결정을 돕는 기능.

### 핵심 요구사항

- **비교 그룹 생성**: "화장실 세면대 후보", "거실 바닥재 후보" 등
- **제품 카드**:
  - 제품명, 브랜드, 가격
  - 사진 (직접 촬영 또는 URL)
  - 장점/단점 메모
  - 쇼룸/구매처 링크
  - 별점/선호도
- **비교 뷰**: 2~4개 제품 나란히 비교 테이블
- **결정 처리**: "이걸로 결정" → 프로젝트 보드 F01 해당 항목에 자동 반영
- **대화형 입력**: "세면대 후보에 OO브랜드 추가해줘, 가격은 50만원"

## F09. 시공업자 관리

시공업자/전문가 정보 관리 및 견적 비교.

### 핵심 요구사항

- **업자 프로필**:
  - 이름/업체명, 전화번호, 이메일
  - 전문 분야 (배관, 전기, 타일, 도배 등)
  - 메모, 평가, 별점
- **견적 관리**:
  - 견적서 파일 첨부 (PDF/이미지)
  - 견적 금액 기록
  - 여러 업자 견적 비교 테이블
- **이력 관리**: 연락 기록, 방문 일정, 작업 완료 여부
- **장소 탐색 F02 연결**: 검색에서 찾은 업자를 바로 등록
- **프로젝트 관리 F03 연결**: 업자를 태스크에 배정

## F10. 영감 보드 (무드보드)

인테리어 레퍼런스 이미지/링크를 공간별로 수집하고 정리.

### 핵심 요구사항

- **수집 방식**:
  - 이미지 직접 업로드
  - URL 붙여넣기 (인스타, 핀터레스트, 블로그 등) → 썸네일 자동 생성
  - 대화형: "이 사진 화장실 무드보드에 추가해줘"
- **정리**: 공간별/태그별 분류, 드래그앤드롭 정렬
- **메모**: 각 이미지에 "이 느낌의 타일", "이 색감" 등 메모
- **공유**: 시공업자에게 "이런 느낌으로 해주세요" 레퍼런스로 공유 가능
- **프로젝트 보드 연결**: 해당 공간/항목에서 무드보드 바로 조회

---

## F11. 계약/문서 보관

시공 관련 계약서, 견적서, 보증서 등 문서를 안전하게 보관.

### 핵심 요구사항

- **문서 업로드**: PDF, 이미지, Word 등
- **메타데이터**: 문서 종류 (계약서/견적서/보증서/기타), 업체명, 날짜, 메모
- **카테고리별 정리**: 문서 종류별, 업체별, 날짜별 필터
- **검색**: 문서명, 업체명으로 검색
- **Supabase Storage**: 파일 저장
- **시공업자 관리 F09 연결**: 해당 업자 프로필에서 관련 문서 바로 조회
