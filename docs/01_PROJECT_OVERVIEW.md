# 🏠 집짓기 에이전트 (House Building Agent)

## 프로젝트 개요

집 인테리어/시공 프로젝트의 전 과정을 관리해주는 AI 기반 웹앱.
결정사항 정리, 쇼룸/매장 탐색, 동선 계획, 시공 관리, 비용 추적까지 한 곳에서.

## 배경

- 포천 한옥 인테리어 프로젝트 진행 중
- 공간별(화장실, 거실, 방 등) × 항목별(바닥, 벽, 세면대 등) 결정사항이 많음
- 결정한 것들을 정리하고, 쇼룸 방문하고, 시공업자 찾고, 비용 관리하는 과정이 복잡
- 이 모든 걸 하나의 앱에서 대화형 + 직접 입력으로 관리하고 싶음

## 사용자

- **성훈** — 프론트엔드 개발자, 기술적 의사결정 담당
- **태수** — 공동 운영자, 함께 인테리어 결정 및 실행

## 핵심 가치

1. **대화형 입력** — "화장실 세면대는 분청으로 결정했어" 하면 자동으로 보드에 반영
2. **직접 입력** — 클릭/타이핑으로 직접 항목 편집도 가능
3. **직접 보러 가기 지원** — 쇼룸/매장 찾기, 동선 계획, 전화번호/영업시간 제공
4. **모든 정보 한 곳에** — 결정사항, 비용, 사진, 문서, 업자 정보 전부 통합

## 기술 스택

- **Frontend**: Next.js (App Router) + TypeScript + Tailwind CSS + shadcn/ui
- **Backend/DB**: Supabase (PostgreSQL + Auth + Storage)
- **AI**: Claude API (대화형 입력, Q&A, Notion 분석, 영수증 OCR)
- **지도/장소**: Naver Maps API 또는 Kakao Maps API
- **Notion 연동**: Notion API
- **배포**: Vercel
